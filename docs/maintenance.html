Overview

🔧 Under Maintenance

NomChat is currently undergoing a major rework. Some services may be temporarily unavailable.

⚠️ Service temporarily unavailable

The main NomChat platform is offline while we rebuild it from the ground up. We appreciate your patience during this time.

What's happening?

We're doing a complete overhaul of the NomChat platform — new architecture, improved performance, and a redesigned experience across all connected services. This is a major rework that touches nearly every part of the system.

During this period, the main NomChat interface is not accessible. We're working as fast as we can to bring everything back online better than before.

What still works?

The following services remain fully operational during maintenance:

🔑

NomChat ID Auth API

Authentication endpoints are live. Apps can still sign users in.
💳

hQd Pays

Payment infrastructure is unaffected and processing normally.
🤖

AI Chat Company

AI chat services are running independently and fully available.

Timeline

We don't have a fixed date yet — we'd rather take the time to do this right than rush a broken release. Follow our updates for the latest news.

📢 Coming soon

The new NomChat will launch with a redesigned interface, faster performance, and new features. Stay tuned.

Need help?

If you have an urgent issue or need access to your account data, the NomChat ID authentication API is still available. See the Authentication API docs for details.
